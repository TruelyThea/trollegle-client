Lurk Rate: Lurk rate is how often the lurk message is sent during inactivity to keep from being kicked. It should be a positive integer. If it is less than 60, it is a number of minutes. Otherwise, if it is less than 3541, it is a number of seconds. Otherwise it is a number of milliseconds.

Lurk Message: The value should be a list of commands and messages separated by the pipe (|) character, one of which will be randomly sent when every %lt lurk rate > of inactivity.

Loging to an output file: Chosen: Output Filename: :today: will be replaced with today's date (UTC time). Other variables surrounded by colons may be filled in the query string. For instance if I put :room:-:tooday:.log as the filename, I could fill in the :room: in the query string of the /-loadrc command with something like ?room=bellawhiskey.
